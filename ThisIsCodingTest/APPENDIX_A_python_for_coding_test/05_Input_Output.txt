5. 입출력

일고리즘 문제 풀이의 첫 번째 단계는 데이터를 입력받는 것이다.

입력 예시
5
65 90 75 34 99

출력 예시
99 90 75 65 34

파이썬은 데이터를 입력받을 때 input()을 이용한다. input()의 경우 한 줄의 문자열을 입력받도록 해준다. 만야 파이썬에서 입력받은 데이터를 정수형
데이터로 처리하기 위해서는 int()를 사용해주어야 한다.

입력데이터는 공백으로 구분되어 입력되는 경우가 많다. 그래서 입력받은 문자열을 띄어쓰기로 구분하여 각각 정수 자료형의 데이터로 저장하는 코드를 자주
사용한다.

list(map(int, input().split()))

간혹 문제에서 매우 많은 수의 데이터가 연속적으로 입력되곤 하는데, 이 때 입력을 빠르게 받는 방법을 알고 있어야 한다. C++의 경우 cin 보다 scanf를
사용하는 것을 권장한다. 파이썬의 경우에는 sys 라이브러리에 정의되어 있는 sys.stdin.readline() 함수를 이용한다.

-----------------------------------------------------------------------------------------------------------------------
import sys
sys.stdin.readline().rstrip()
-----------------------------------------------------------------------------------------------------------------------

파이썬에서 출력에 주의해야 할 것은 변수의 자료형에 유의해야 한다. 정수형을 출력하고 싶다면 str()을 사용해 문자열 데이터로 바꿔주어야 하거나
f-string 문법을 사용해주자.

-----------------------------------------------------------------------------------------------------------------------
answer = 7

print("정답은 " + str(answer) + "입니다.")  # 정답은 7입니다.

print("정답은", str(answer), "입니다.")  # 정답은 7 입니다.

print(f"정답은 {answer}입니다.")  # 정답은 7입니다.
-----------------------------------------------------------------------------------------------------------------------
