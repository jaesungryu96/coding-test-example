4. 함수

함수는 프로그래밍을 하다 보면, 똑같은 코드가 반복적으로 사용되어야 할 경우 큰 도움이 된다. 함수를 사용하지 않으면 똑같은 소스코드를 매번 일일이
작성해야 하므로, 소스코드가 길어지며 이로 인해 프로그램의 크기가 비효율적으로 커진다.

-----------------------------------------------------------------------------------------------------------------------
def 함수명(매개변수):
    실행할 소스코드
    return 반환 값
-----------------------------------------------------------------------------------------------------------------------

함수를 호출하는 과정에서 인자(Argument)를 넘겨줄 때, 파라미터의 변수를 직접 지정해서 값을 넣을 수 있다. 그리고 함수 안에서 함수 밖의 변수
데이터를 변경해야 하는 경우가 있는데, 이 때는 함수에서 global 키워드를 사용하면 된다. 끝으로 파이썬에서는 람다 표현식(Lambda Express)을 사용할 수 있다.
람다 표현식을 이용하면 함수를 매우 간단하게 작겅하여 적용할 수 있다.

-----------------------------------------------------------------------------------------------------------------------
# 일반적인 함수
def add(a, b):
    return a + b

print(add(3, 7))

print((lambda a, b: a + b)(3, 7))  # 람다 표현식
-----------------------------------------------------------------------------------------------------------------------
