3. 반복문

반복문은 특정한 소스코드를 반복적으로 실행하고자 할 때 사용할 수 있다. while, for 문이라는 두 가지 종류의 반복문이 있다.

while 문

while 문은 조건식이 참일 때에 한해서, 반복적으로 수행된다(조건식이 참인 동안). 항상 참인 조건식을 설정하면 무한 루프(Infinite Loop)가 발생하니
주의하도록 하자.

-----------------------------------------------------------------------------------------------------------------------
i = 1
result = 0

while i <= 9:
    result += i
    i += 1

print(result)  # 45
-----------------------------------------------------------------------------------------------------------------------

for 문

반복문으로 for 문을 이용할 수도 있다. 리스트를 사용하는 대표적인 for 문의 구조는 다음과 같다. in 뒤에 오는 데이터로는 리스트, 튜플, 문자열 등이 사용될 수 있다.

-----------------------------------------------------------------------------------------------------------------------
for 변수 in 리스트:
    실행할 소스코드
-----------------------------------------------------------------------------------------------------------------------
