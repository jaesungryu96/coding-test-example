2. 조건문

조건문은 프로그램의 흐름을 제어하는 문법이다.

-----------------------------------------------------------------------------------------------------------------------
score = 85

if score >= 90:
    print('학점 A')
elif score >= 80:
    print('학점 B')
elif score >= 70:
    print('학점 C')
else:
    print('학점 F')
-----------------------------------------------------------------------------------------------------------------------

비교 연산자

조건문에서는 비교 연산자를 자주 사용한다.

X == Y      X와 Y가 서로 같을때 참이다.
X != Y      X와 Y가 서로 다를때 참이다.
X > Y       X가 Y보다 클때 참이다.
X < Y       X가 Y보다 작을때 참이다.
X >= Y      X가 Y보다 크거나 같을때 참이다.
X <= Y      X가 Y보다 작거나 같을때 참이다.

논리 연산자 (Logical Operators)

파이썬에는 3가지 논리 연산자가 있다.

X and Y     X와 Y가 모두 참일 때 참이다.
X or Y      X와 Y 중에 하나만 참이면 참이다.
not X       X가 거짓일 때 참이다.

파이썬의 기타 연산자

파이썬은 'in 연산자'와 'not in 연산자'를 제공한다. 여러 개의 데이터를 담는 리스트, 튜플, 딕셔너리 같은 자료형에 특정 데이터 값이 존재하는지
확인하는 연산이 필요할 때 사용한다.

X in 리스트        리스트 안에 X가 들어가 있을 때 참이다.
X not in 문자열    문자열 안에 X가 들어가 있지 않을 때 참이다.

-----------------------------------------------------------------------------------------------------------------------
score = 85

result = "Success" if score >= 80 else: "Fail"

a = [1, 2, 3, 4, 5, 5, 5]

remove_set = {3, 5}

result = [i for i in a if i not in remove_set]  # [1, 2, 4]

x = 15

if x > 0 and x < 20: 이 문장은 if 0 < x < 20: 과 똑같다.
